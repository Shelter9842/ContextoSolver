# Contexto Solver

Find the top word in Contexto games with just a word and an index pair.

## Installation

1. Clone the repository:
  ```bash
   git clone https://github.com/yourusername/contexto-solver.git
   cd contexto-solver
  ```

2. Install the required dependencies using pip:
  ```
  pip install -r requirements.txt
  ```

3. Run the script:
  ```
  python index.py
  ```

## Usage

```
 > Enter one or more word-index pairs (e.g., 'house 40 castle 50'): *word here* *index here*
```
